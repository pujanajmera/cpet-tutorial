Welcome to Lumache's documentation!
===================================

**CPET** (Computation of Protein Electric Field Topologies) is a Python-based package designed for calculations and analysis of electric fields, their geometries, 
electrostatic potentials, and dynamics in enzymes

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
